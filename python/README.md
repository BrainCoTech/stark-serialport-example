# Python

## Requirement

- Python 3.8~3.12
- Mac 10.15 or later
- Windows 10 build 10.0.15063 or later
- Ubuntu 20.04 LTS or later

## Usage

```shell
cd python

# 安装依赖
pip3 install -r requirements.txt
```
